# O que arrastar para a pasta _ParaExportar

Arraste manualmente para `Assets/_ParaExportar` os seguintes itens essenciais do seu projeto:

1. Scripts/Core/           (arquitetura nova, base OOP)
2. Scripts/                (outros scripts importantes: TaskSystem, NPCController, etc.)
3. Prefabs/                (prefabs de Player, NPC, Forno, Caixa, etc.)
4. Models/                 (modelos 3D)
5. Scenes/                 (cenas que você quer reaproveitar)
6. TextMesh Pro/           (se usar na UI)
7. Input System arquivos:
   - InputSystem_Actions.cs
   - InputSystem_Actions.inputactions
   - InputSystem.inputsettings.asset
8. Outros assets importantes (materiais, animações, texturas, etc.)

Depois de arrastar tudo, clique com o botão direito na pasta `_ParaExportar` e escolha "Export Package..." para criar o pacote para o novo projeto.

Se tiver dúvida sobre algum item, pergunte aqui antes de exportar!
